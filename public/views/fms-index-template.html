Loading ThingSPIN FMS

If you're seeing this ThingSPIN FMS has failed to load its application files

1. This could be caused by your reverse proxy settings.

2. If you host thingspin under subpath make sure your thingspin.ini root_url setting includes subpath

3. If you have a local dev build make sure you build frontend using: npm run dev, npm run watch, or npm run build

4. Sometimes restarting thingspin-server can help
